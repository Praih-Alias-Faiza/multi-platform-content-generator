# 🚀 Multi-Platform Content Generator

An AI-powered app built with Google AI Studio and 
Gemini AI that generates ready-to-post captions and 
hashtags for multiple platforms simultaneously.

## 🔗 Live App
https://multipost-ai-40456361814.asia-southeast1.run.app

## ✨ Features
- Multi-platform content generation in one click
- Tone selector — Professional, Funny, Inspirational & Casual
- Copy All button for each platform
- Clean purple & pink gradient UI

- Gemini AI
- No traditional coding required
